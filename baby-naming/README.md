# Baby Name Tinder

Mobile-first Web-App für werdende Eltern zur spielerischen Namensfindung.

## Features

- Swipe-/Button-Flow für Babynamen
- Aktionen: Nein, Like, Super Like, Super Duper Like
- LocalStorage-Speicherung, kein Backend
- Lernlogik: bevorzugte Kategorien werden höher gerankt
- Fortschritt, Meilensteine und Session-Ziele
- Export/Import als JSON-Datei
- Partner-Matching mit Top Matches, perfekten Matches und einseitigen Favoriten
- Externe Links zu Google und ChatGPT-Prompt pro Name

## Lokal starten

```bash
npm install
npm run dev
```

Dann im Browser öffnen: `http://localhost:5173`

## Build

```bash
npm run build
```

Der statische Build liegt danach in `dist/`.
